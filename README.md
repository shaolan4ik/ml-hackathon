# hakaton
В проекте содержиться файл Jupiter notebook с описанием и проект в папке ml-hakaton, в котором изменены следующие файлы:  
- hackaton/train/training.py - добавлена логика обработки данных и обучение модели  
- hackaton/service/app.py - добавлен инференс модели, загрузка артефактов в prepere и построение признаков для обращение к модели  
- docker-compose.yam - добавлены ограничения по ресурсам и добавлен проброс папки data/ во внутрь контейнера
